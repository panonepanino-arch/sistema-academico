# Sistema Acadêmico

## Descrição
Este projeto é um sistema acadêmico simples desenvolvido em Python.

O objetivo do sistema é permitir o gerenciamento básico de alunos dentro de uma instituição de ensino.

## Funcionalidades
- Cadastro de alunos
- Listagem de alunos

## Tecnologias utilizadas
- Python
- Git
- GitHub

## Autor
Projeto desenvolvido para fins acadêmicos.
